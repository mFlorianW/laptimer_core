# LaptimerCore

## Build LaptimerCore
There two configure and build presets for CMake. 
Call one the in the project root directory to build the LaptimerCore.

### Debug 
#### Configure
```console
cmake --build --preset debug
```

#### Build
```console
cmake --build --preset debug
```

### Release
#### Configure
``` console
camke --preset release
```

#### Build
``` console
camke --build --preset release
```
